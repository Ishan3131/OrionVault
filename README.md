# OrionVault

OrionVault is a hub of games that allows users to discover and explore titles using structured data.

## Features
* Search games by:
  * Name
  * Genre
  * Creator
  * Price
  * Release date
  * Platform
* Sort games based on:
  * Price
  * Release date
  * Rating
* Filter games based on:
  * Platform
  * Genre
  * Creator
  * Other available metadata
### Data Source

This project uses the RAWG Video Games Database API to fetch game data:

https://rawg.io/apidocs

The API provides information such as game details, ratings, release dates, platforms, and more.
